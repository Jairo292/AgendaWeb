# Organiza Pro

**Organiza Pro** es una aplicación web de gestión de tareas, pendientes y notas personales que funciona completamente en tu navegador. Utiliza `localStorage` para guardar todos tus datos, lo que significa que no requiere un servidor ni conexión a internet para funcionar. Es rápida, privada y fácil de usar.

## ✨ Características

*   **Gestión de Tareas**: Crea tareas con título, detalles, prioridad y un color personalizado.
*   **Calendario Visual**: Un calendario interactivo que muestra tus tareas y te permite filtrar por día.
*   **Fechas y Recurrencia**: Asigna fechas específicas o establece reglas de recurrencia (semanal, mensual, anual).
*   **Listas Inteligentes**: Visualiza tus tareas organizadas en "Hoy", "Próximos 7 días", "Futuro" y "Atrasados".
*   **Notas Rápidas**: Un espacio para apuntes y recordatorios que no tienen una fecha específica.
*   **Búsqueda Integrada**: Encuentra rápidamente cualquier tarea o nota por su título o detalles.
*   **Temas**: Elige entre tema claro, oscuro o automático según la configuración de tu sistema.
*   **Importar/Exportar**: Respalda y restaura todos tus datos en un archivo JSON.
*   **Autolimpieza**: Configura la aplicación para que elimine automáticamente tareas antiguas y completadas.

## 🚀 Cómo usarlo

Este proyecto no necesita compilación ni dependencias. Simplemente sigue uno de estos pasos:

1.  **Abrir el archivo localmente**: Descarga o clona el repositorio y abre el archivo [`index.html`](index.html) directamente en tu navegador web (como Chrome, Firefox, o Edge).
2.  **Servidor de desarrollo local (Opcional)**: Si eres desarrollador, puedes usar cualquier servidor web estático para servir los archivos. Por ejemplo, con la extensión "Live Server" de VS Code.

Toda la lógica de la aplicación se encuentra en [`assets/js/app.js`](assets/js/app.js) y los estilos en [`assets/css/styles.css`](assets/css/styles.css).
